 ============================================== users - country ==============================================
 Keyword: Laravel Using Multiple Database Connections
- Tạo database users và country
- Checkout user_country
- Vào phần config database và .env để cấu hình thêm 1 database mới đó là mysql2
- Tạo thêm migration countries connection đến mysql2: 
  php artisan make:migration create__countries__table --create=countries
- Chạy migration: php artisan migrate
  Thêm dữ liệu vào cho các bảng countries và users
- Query Builder: Truy cập vào mỗi DB bằng cách sử dụng phương thức connection() của DB
- Tạo thêm 1 Model mới có tên là Country: php artisan make:model Country
- Eloquent: Kết nối với DB bằng cách set giá trị cho biến connection trong Model
- setConnection: Kết nối với DB bằng cách sử dụng phương thức setConnection()
